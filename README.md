## habit_app
**Descripción:**  
Aplicación móvil para ayudar a los usuarios a crear y mantener hábitos saludables.

**Contribuciones:**  
- Diseñé la interfaz de usuario, estableciendo la paleta de colores y el estilo visual general.

Dart (Flutter), Android Studio
